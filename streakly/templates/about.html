About Streakly

Streakly is a habit-tracking app developed as our collaborative hackathon project for the Code Institute's 4-month bootcamp in full-stack web development. Built with a modern tech stack including HTML, CSS, Bootstrap, Django, Python, and PostgreSQL, Streakly features a responsive and user-friendly interface. Django and PostgreSQL ensure a robust and scalable backend. Developed collaboratively with agile methodologies, Streakly empowers users to build and maintain habits effortlessly, offering a visually appealing and intuitive platform to track progress and grow streaks. It’s not just an app—it’s the result of dedication, collaboration, and a passion for creating meaningful solutions. Start building your streaks today with Streakly!
